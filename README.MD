Windows 10X skin for windows 11!

Thanks for viewing this, this was my project to continue Win11-Valley10x project, it was edit a lot by myself, so i fork it, because i still use some code and modify some, so please don't download from Main, because that's original code, please download from release, the installation step here:
+ step 1: extract to a folder

+ step 2: install windhawk and install all mods: Windows 11 Start Menu Styler, Windows 11 Taskbar Styler, Taskbar height and icon size, and Taskbar Clock Customization

+ step 3: copy all code to correct place of mods

+ step 4: to the action center logo works, install the font inside the zip

+ step 5: apply desktop walpaper

+ step 6: install open shell from here:https://open-shell.github.io/Open-Shell-Menu/ and apply the custom orb

+ step 7: change the open shell control settings to default windows start menu instead custom open shell one

Requirement(IMPOTANT):
* Old Classic windows 11 start menu(if you had the new one from win11 25h2, download vivetool and run this command vivetool /disable /id:47205210 source:https://onlinecomputertips.com/support-categories/windows/remove-windows-11-25h2-start-menu/)
* Enable Show more suggestion for the start menu, don't worry, it won't make the start menu pinned section smaller
* Windhawk installed if not download from https://windhawk.net/
* In windhawk install following mods: Windows 11 Start Menu Styler, Windows 11 Taskbar Styler, Taskbar height and icon size, and Taskbar Clock Customization
